# Git Cola Icons

## Guidelines

1. Size: 22x22px (the default action icon size of KDE and GNOME)
2. Margin: 1px (20x20px drawing area)
3. Colors: #000000 black, #ffffff white, #cc0000 red, #00dd26 green, #1961ff blue
